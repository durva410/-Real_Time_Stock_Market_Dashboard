Real-Time Stock Market Dashboard

A simple and interactive Stock Market Dashboard built using Streamlit, Plotly, and yFinance.
This app allows users to visualize real-time stock data with candlestick charts, volume analysis, and key metrics.

Features
- Real-time stock data fetching
- Interactive candlestick chart
- Volume analysis chart
- Key metrics (Latest Price, Change, % Change)
- Raw data table view
- User input for:
   Stock symbol
   Time period
   Data interval


Tech Stack
- Python
- Streamlit
- yfinance
- Pandas
- Plotly
